我將助教提供的SocketSample加以修改，MainActivity為Client端，我將Lab10的calculator功能加入後，把計算結果給client再傳到server端。在Main中我加入了JFrame的視窗顯示。而在Server中，extends了JFrame以產生server的GUI介面，當client所輸入的IP與真正的IP相符時，會進到計算機的畫面，當答案計算出來後server則會顯示結果。
額外功能：ProgressBar的顯示，我在Server多加入了progressbar，把client計算出的答案傳入server後以bar的方式顯示出來，若有小數點，是以無條件捨去的方式顯示，範圍設定在0至100之間。
加入ProgressBar時遇到的困難：因為client傳給server的是一串字串，而我需要的答案是在字串中等號後面的結果，因此我在server中用substring把後方的數字擷取出來，再更新progressbar的值。
